1. Global Scope

 Variables accessible everywhere in the program

2. Script Scope

 Top-level variables (let/const) that are global but not attached to the window object

3. Local Scope (Function Scope)

 Variables accessible only inside the function where they are defined

4.  Block Scope

 Variables (let/const) accessible only inside a {} block

5.  Lexical Scope

 Scope determined by where variables are written in code, not where they are called

Global → everywhere
Script → global but not window
Local → inside function
Block → inside {}
Lexical → based on code structure
