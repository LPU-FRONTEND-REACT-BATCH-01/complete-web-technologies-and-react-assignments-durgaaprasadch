


-------------------------------------------------------------------------------------------------------------------------------------
A ternary operator is a shortcut for if-else.

condition ? value_if_true : value_if_false

----------------------------------------------------------------------------------------------------------------------------------

Normal if-else:
let age = 18;

if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

--------------------------------------------------------------------------------------------------------------------------------
Same using ternary:
let age = 18;

let result = age >= 18 ? "Adult" : "Minor";
console.log(result);
